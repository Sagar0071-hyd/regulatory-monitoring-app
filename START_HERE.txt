1. Install Node.js & MongoDB (if not already)
   https://nodejs.org/
   https://www.mongodb.com/

2. Backend Setup:
   cd regulatory-monitoring-app/backend
   npm install
   cp .env.example .env
   # Update .env with your credentials
   npm run dev

3. Frontend Setup (New Terminal):
   cd regulatory-monitoring-app/frontend
   npm install
   npm start
