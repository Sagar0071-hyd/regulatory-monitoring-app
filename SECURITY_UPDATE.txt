⚠️ IMPORTANT SECURITY STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before Deploying to Production:

1. Change Admin Password
   • Do NOT use default password
   • Use strong password with special characters
   • Update in backend/.env

2. Update Other Secrets
   • JWT_SECRET: Use openssl rand -base64 32
   • RAZORPAY_KEY_ID: Use production keys
   • RAZORPAY_KEY_SECRET: Use production secret
   • EMAIL_PASSWORD: Use app password, not Gmail password

3. Verify .gitignore
   • Check .env is ignored
   • Check ADMIN_CREDENTIALS.md is ignored
   • Verify before pushing to GitHub

4. Test Credentials
   • Verify login works with new credentials
   • Test password reset
   • Confirm admin panel access
